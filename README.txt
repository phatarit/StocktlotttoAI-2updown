
StockLottoAI 5 หลัก + AI

วิธีใช้:
1. ติดตั้งไลบรารี
   pip install -r requirements.txt

2. เปิดแอป
   streamlit run app.py

3. วางเลขหวยหุ้น 5 หลัก (บรรทัดละ 1 ชุด) แล้วดูผลวิเคราะห์และ AI ทำนายเลขหลักสุดท้าย

- Logic ปกติ: วิเคราะห์เลขเด่น, สองตัว, สามตัว จากสถิติ
- AI (ML): ใช้ Neural Network ทำนายเลขหลักสุดท้าย
